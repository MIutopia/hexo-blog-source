## 前言

Nginx 是一款高性能的 HTTP 和反向代理服务器，广泛应用于 Web 服务、负载均衡和反向代理。这篇文章记录了在 CentOS 上从源码编译安装 Nginx 的完整过程。

## 安装依赖

```bash
yum -y install gcc gcc-c++ automake pcre pcre-devel zlib zlib-devel openssl openssl-devel
```

## 下载并解压 Nginx

```bash
# 创建目录
mkdir /opt/nginx
cd /opt/nginx

# 下载 Nginx
wget http://nginx.org/download/nginx-1.26.1.tar.gz

# 解压
tar -xvf nginx-1.26.1.tar.gz
```

> 官网下载地址：[http://nginx.org/en/download.html](http://nginx.org/en/download.html)

## 编译安装

```bash
# 创建安装目录
cd /opt/nginx
mkdir nginx-1.26.1_install

# 进入源码目录
cd nginx-1.26.1

# 配置（指定安装目录）
./configure --prefix=/opt/nginx/nginx-1.26.1_install

# 编译
make

# 安装
make install
```

### 安装后的目录结构

| 目录 | 说明 |
|------|------|
| `conf/` | Nginx 配置文件（nginx.conf） |
| `html/` | 默认网页文件 |
| `logs/` | 日志文件 |
| `sbin/` | Nginx 主程序 |

## 启动 Nginx

```bash
# 检查配置语法
/opt/nginx/nginx-1.26.1_install/sbin/nginx -t

# 启动 Nginx
/opt/nginx/nginx-1.26.1_install/sbin/nginx

# 查看版本
./sbin/nginx -v

# 查看进程
ps -ef | grep nginx
```

## 常用命令

```bash
# 停止 Nginx
/opt/nginx/nginx-1.26.1_install/sbin/nginx -s stop

# 重启 Nginx
/opt/nginx/nginx-1.26.1_install/sbin/nginx -s reopen

# 重新加载配置
/opt/nginx/nginx-1.26.1_install/sbin/nginx -s reload
```

## 验证

```bash
# 关闭防火墙（测试用）
systemctl stop firewalld
systemctl disable firewalld
```

访问 `http://192.168.10.100:80` 即可看到 Nginx 默认欢迎页面。

## 常见问题

### ❌ 端口被占用

**原因**：80 端口已被其他服务占用。

**解决**：

```bash
# 查看占用端口的进程
netstat -tlnp | grep 80

# 或修改 Nginx 监听端口
vi /opt/nginx/nginx-1.26.1_install/conf/nginx.conf
# 修改 listen 80; 为其他端口
```

### ❌ 权限不足

**原因**：非 root 用户启动 Nginx。

**解决**：使用 `sudo` 或切换到 root 用户。
