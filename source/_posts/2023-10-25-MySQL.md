## 前言

MySQL 是最流行的关系型数据库管理系统之一，广泛应用于 Web 应用开发。这篇文章记录了在 CentOS 上安装 MySQL 的完整过程，以及 MariaDB 作为替代方案的说明。

## 安装前检查

```bash
# 检查系统是否已安装 MySQL
rpm -qa | grep mysql

# 如果已安装，卸载旧版本
rpm -e mysql                    # 普通删除
rpm -e --nodeps mysql           # 强制删除（忽略依赖）
```

## 安装 MySQL

### 方法一：使用 YUM 安装

参考 [MySQL 官方文档](https://dev.mysql.com/downloads/repo/yum/)

```bash
# 下载 MySQL 仓库
wget http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm

# 安装仓库
rpm -ivh mysql-community-release-el7-5.noarch.rpm

# 更新并安装
yum update
yum install mysql-server

# 设置权限
chown -R mysql:mysql /var/lib/mysql/

# 初始化
mysqld --initialize

# 启动 MySQL
systemctl start mysqld

# 检查状态
systemctl status mysqld
```

> **注意**：第一次启动 MySQL 服务时，服务器会自动进行初始化配置。

### 方法二：使用 MariaDB（MySQL 替代方案）

MariaDB 是 MySQL 的一个分支，主要由开源社区维护，采用 GPL 授权许可。开发这个分支的原因之一是：甲骨文公司收购了 MySQL 后，有将 MySQL 闭源的潜在风险。

MariaDB 的目的是完全兼容 MySQL，包括 API 和命令行，可以轻松成为 MySQL 的代替品。

```bash
# 安装 MariaDB
yum install mariadb-server mariadb 

# 启动/停止/重启
systemctl start mariadb
systemctl stop mariadb
systemctl restart mariadb

# 设置开机启动
systemctl enable mariadb
```

## 验证安装

```bash
# 查看版本
mysqladmin --version
# 输出示例：mysqladmin  Ver 8.42 Distrib 5.6.51, for Linux on x86_64

# 如果以上命令未输出任何信息，说明 MySQL 未安装成功

# 进入 MySQL
mysql
```

## 常见问题

### ❌ 忘记 root 密码

**解决**：

```bash
# 停止 MySQL 服务
systemctl stop mysqld

# 以安全模式启动
mysqld_safe --skip-grant-tables &

# 进入 MySQL 修改密码
mysql
> USE mysql;
> UPDATE user SET password=PASSWORD('新密码') WHERE user='root';
> FLUSH PRIVILEGES;
> EXIT;

# 重启 MySQL
systemctl restart mysqld
```

### ❌ 远程连接被拒绝

**原因**：MySQL 默认只允许本地连接。

**解决**：

```sql
-- 授权远程访问
GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY '密码' WITH GRANT OPTION;
FLUSH PRIVILEGES;
```
