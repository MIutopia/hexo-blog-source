在 Nginx 的配置中，虚拟主机是一个重要概念，它允许单个 Nginx 服务器为多个域名或 IP 地址提供服务，而不需要为每个域名或 IP 地址运行单独的 Nginx 实例。

## 创建项目路径

```bash
# 创建项目目录
mkdir -p /var/www/site1
mkdir -p /var/www/site2
```

## 配置虚拟主机

编辑 Nginx 配置文件 `/etc/nginx/nginx.conf` 或创建单独的配置文件：

```nginx
server {
    listen 80;
    server_name site1.example.com;
    root /var/www/site1;
    index index.html;
}

server {
    listen 80;
    server_name site2.example.com;
    root /var/www/site2;
    index index.html;
}
```

## 重启 Nginx

```bash
nginx -t          # 测试配置
systemctl restart nginx   # 重启 Nginx
```
